# VibeDesk Trailer (Remotion)

## Setup
```sh
cd remotion
npm install
```

## Preview
```sh
npm run dev
```

## Render
```sh
npm run render
```

## Assets
Place assets in `remotion/public/trailer/`:
- `logo.png`
- `screen.png`
- `music.mp3` (required)

## Notes
- Format: 1920x1080 (16:9)
- Duration: 40s
- No voiceover, text-only trailer
